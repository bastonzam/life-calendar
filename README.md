# Life Calendar — Features (React)

ส่วน "Features" ที่ได้แรงบันดาลใจจากแนวคิด Life Calendar เขียนใหม่ด้วย React โดยไม่คัดลอกข้อความ/รูป/โค้ดจากเว็บไซต์ต้นทาง

## วิธีเริ่มต้น

```bash
npm install
npm run dev
```

เปิดเบราว์เซอร์ที่ URL ที่ Vite แจ้ง (เช่น `http://localhost:5173`).

## โครงสร้าง

```
life-calendar-feature-react/
├─ public/
│  └─ favicon.svg
├─ src/
│  ├─ components/
│  │  └─ FeatureSection.jsx
│  ├─ App.jsx
│  ├─ main.jsx
│  └─ styles.css
├─ index.html
├─ package.json
└─ vite.config.js
```

## หมายเหตุ
- โปรเจ็กต์นี้สร้างขึ้นเพื่อ “เลียนแนวคิด” ส่วนฟีเจอร์เท่านั้น ไม่คัดลอกเนื้อหา/ดีไซน์เฉพาะเจาะจงของเว็บไซต์อื่น
- คุณสามารถปรับข้อความ/สไตล์/แอนิเมชัน เพิ่มไอคอนหรือการจัดวางให้ใกล้ mood & tone ที่ต้องการได้
